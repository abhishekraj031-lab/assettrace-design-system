# Logo

The AssetTrace logo is available in 4 variants. All variants should be used as SVG files exported from Figma.

---

## Variants

| Variant | File | Background |
|---|---|---|
| Black | `assets/logos/logo-black.svg` | Light backgrounds |
| White | `assets/logos/logo-white.svg` | Dark backgrounds |
| Horizontal | `assets/logos/logo-horizontal.svg` | Wide layouts, headers |
| Stacked | `assets/logos/logo-stacked.svg` | Square spaces, app icons |

---

## Usage

**Do**
- Use the Black logo on white or light grey backgrounds
- Use the White logo on dark, brand-coloured, or image backgrounds
- Always maintain the minimum clear space around the logo (equal to the height of the logomark)

**Don't**
- Don't stretch, rotate, or distort the logo
- Don't place the logo on busy or low-contrast backgrounds
- Don't recreate or redraw the logo — always use the exported SVG

---

## Code

```html
<!-- Black logo — for light backgrounds -->
<img src="assets/logos/logo-black.svg" alt="AssetTrace" height="40">

<!-- White logo — for dark backgrounds -->
<img src="assets/logos/logo-white.svg" alt="AssetTrace" height="40">
```

---

## Exporting from Figma

1. Open the Figma file → go to the **Logos** page
2. Select the logo variant you want
3. In the right panel → click **Export**
4. Choose **SVG** format → click **Export**
5. Save to `assets/logos/` in this repository
